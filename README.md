# SAdashboard
SPA CMS for StudentAvatar.

## Getting Started
Required:
* [Node.js](https://nodejs.org/en/)
* [Firebase CLI](https://www.npmjs.com/package/firebase-tools)

Authenticate:
```
$ firebase login
```
Clone repository
```
$ git clone https://github.com/gunatilake/SAdashboard.git
```
Deploy locally
```
$ firebase serve
```
